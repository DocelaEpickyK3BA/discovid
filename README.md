## Installation
Remember you need to install [Node.js](https://nodejs.org/en/)(recommended version) to run DisCovid on your computer.

1. Install dependencies 
```bash
npm i
```

2. Create .env file and save these variables in
```dotenv
TOKEN=yourtokenhere
PREFIX=yourprefixhere
```

3. Run the bot!
```bash
npm start
```
